Explore Competitive Catabolism

An interactive simulator for the multi nutrient model

About This Simulator

This web application provides interactive simulations for fasted metabolite dynamics. Explore how different nutrients (glucose, fatty acids, lactate, and ketones) compete for oxidation under various physiological and pathological conditions. The simulations are based on a validated computational model (presented here) and allows you to manipulate parameters to explore different scenarios.
